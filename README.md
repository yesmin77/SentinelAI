# SentinelAI – AI Powered Crypto Scam Detector

SentinelAI is an AI assistant designed to improve security in the Binance ecosystem.

It helps users detect potential crypto scams by scanning:

• Wallet addresses  
• Token contracts  
• Suspicious links  

## Features

- Detect scam wallets
- Identify honeypot tokens
- Flag phishing links
- AI risk scoring system

Users can paste a wallet address or token contract and instantly receive a risk score.

This concept demonstrates how AI can protect Binance users from fraudulent projects and malicious transactions.
